
# Flask Admin Example

```bash
git clone https://github.com/rochacbruno/flask_admin_example
cd flask_admin_example
python3.6 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

flask run
```

Access http://localhost:5000/admin

- Username: admin 
- Password secret
